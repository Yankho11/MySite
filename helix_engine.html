Initialisation settings selector

I have not yet implemented a GUI for a user to use in selecting desired settings, so for the time being and for building purposes, I opted to use the console instead. Once things start moving forward, I do intend on implementing a GUI using ImGui.

Initialisation settings output

The console outputs the desired Initialisation settings afterwards in order to keep you aware of the settings used to initialize the context window.

Fill render default shaders

This is just to show that the program is setup to render something after it is initialized, and the default shaders are used in this render

Fill render custom shaders

This is just to show that there is multi-colour render functionality. This was done using 2 different colors on 4 different triangles, if one wanted they could have a different colour for each triangle.

Wiring render default shaders

The wiring is an option you can toggle while the program is running, for the time being and just for testing, the toggle is bound to the W key

Wiring render custom shaders

Just as with the default shader, the wiring is an option you can toggle while the program is running, for the time and being just for testing, the toggle is bound to the W key
